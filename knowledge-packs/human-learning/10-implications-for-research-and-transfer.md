# Implications for research and a redefinition of transfer

**Research design.** First principles of computation demand that how information is *represented* (categorical, sequential, structural, predictive) governs how it can be learned, stored, accessed, and processed. So rigorous learning research should account for these mechanisms:

- **Separate independent variables, outcomes, mediators, and moderators by information type** — predictive vs sequential vs cause-effect vs categorical — and identify which system will process each.
- **Align assessment with intervention.** None of the systems is inherently "fast" or "slow" and there are no fast/slow information types, so a variable like immediate-vs-delayed feedback only correlates with learning if mediated by a specific system's mechanism. The paper cites a large immediate-vs-delayed-feedback study that yielded a *high-variance null result* — expected when the type of learning, type of feedback, and opportunity to use the feedback aren't distinguished. If feedback promoted deliberative learning but assessment measured procedural learning (or vice versa), gains are underestimated.
- Incorrect assumptions about the brain's information processing can compromise the interpretation of findings ("neuromyths").

**Transfer, redefined by system boundaries.** Transfer (generalizing prior knowledge to a new context) is notoriously hard to define and is usually described by surface features (domain, e.g. math→science; or context, e.g. theory→applied). The model offers a mechanistic taxonomy: **transfer occurs *within* each system, but differently in each:**

- **Perceptual transfer** — generalizing new situations by recognizing them as modified categories (and analogical reasoning).
- **Deliberative transfer** — applying familiar choices to new outcomes, or new choices to familiar outcomes, via causal recognition and mental exploration.
- **Procedural transfer** — integrating previously automated skills into new sequences (well-practiced components recombined).
- **Instinctual transfer** — generalizing predictive cue–outcome associations across contexts.

A taxonomy grounded in **generalization and consolidation** thus gives researchers a neuroscientific language with less conceptual ambiguity and more explanatory depth than surface-feature definitions.
