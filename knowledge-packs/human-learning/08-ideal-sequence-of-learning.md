# The ideal sequence of learning exercises

The information regularities of each system imply an **ideal ordering** of learning activities. A canonical sequence already appears in education — begin with icebreakers/motivation, move to conceptual structure, then to efficient solutions — and it works, the authors argue, because it transitions **instinctual → deliberative → procedural**, each building on the last. The model's key refinement is that **perceptual learning is usually neglected** and should be added **early**, since the action-selection systems cannot function without the Perceptual System.

The recommended order and why:

1. **Instinctual first.** Begin by managing social dynamics and motivation. Students who are socially isolated, bored, or perceive threat face barriers to learning in *every* other system. A heightened threat response even redirects attention *toward* threatening cues and away from the ones the educator intends — so establishing safety/belonging enables all subsequent learning.
2. **Perceptual second** (the refinement). Perceptual categorization is the starting point for all cognition; it should be emphasized early and then **revisited at regular intervals** as students work through the other systems.
3. **Deliberative third.** Place structured, exploratory reasoning *before* procedural learning. Mentally navigating dead ends and inefficient paths builds the knowledge structures needed for later creative decision-making — which is hard to do once an efficient path is already known. Emerging evidence supports deliberative-before-procedural.
4. **Procedural last (the capstone).** Automation causes loss of awareness of the components and active inattention to everything outside the automated chain, which can directly interfere with the other systems' learning. Because it is hard to pull students away from a known efficient solution, procedural automation is best saved for the end.

The same progression is observable across scales — a career (kindergarten attitudes → foundational concepts → efficient professional skills), a semester, and a single lesson. The takeaway: **scaffold curricula so learning in each system complements and supports the others**, rather than optimizing one in isolation.
