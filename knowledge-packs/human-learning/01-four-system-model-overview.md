# The four-system model of human learning

Allen, Redish & Kizilcec ("Fundamental Mechanisms of Human Learning", arXiv:2509.17202v2) integrate 21st-century neuroscience into a unified, mechanistic model of learning. The core claim: learning is an information-processing phenomenon, and decision-making is a three-stage process — perception, action-selection, and execution — with cognition and learning happening in the first two stages.

Neural systems cope with environmental complexity by selectively extracting distinct **types** of information, each handled by a separate system with its own memory store and its own learning mechanism:

- **Perceptual System** — extracts *categorical* relationships (which situations group together); it categorizes the present situation.
- **Instinctual System** — extracts *predictive* relationships (which cue predicts which event); releases innate hereditary actions.
- **Deliberative System** — extracts *causal and structural* relationships (maps of cause-and-effect and abstract ideas); simulates the future.
- **Procedural System** — extracts *sequential* relationships (the order actions occur in); releases well-practiced action chains.

The Perceptual System is a single input feeding the three action-selection systems, which run **in parallel, not in series**, so they must be modeled as separate systems (analogous to eyes/ears/nose as parallel sensory inputs).

Because each system learns differently from different environmental conditions, effective teaching requires **aligning the learning goal and environment with the target system**. This is presented not as a replacement for established models (Bloom's Taxonomy, Deliberate Practice, Cognitive Load Theory, etc.) but as a refinement — those models were "not incorrect, merely incomplete." The framework generalizes well beyond formal education to therapy, athletics, public-health messaging, organizational training, and social life.
